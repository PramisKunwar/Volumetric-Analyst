# Volumetric-Analyst 
This is a chrome extension that helps a chemistry student while studying volumetric analysis by providing then cheat sheets, law of equivalence calculator and Normality/Molarity finder,etc. Aligned with Grade 12 Chemistry

## Features 
- Unit Converter: M, N, g/L, %, ppm, ppb, molality — all interconvertible.
- Titration Solver: N₁V₁ = N₂V₂ with step-by-step working shown.
- Eq. Weight Helper: Acids, bases, salts, redox agents — instant calculation.
- Fully offline: No data leaves your browser. 

## Motivation 
This project is inspired from grade 12 Chemistry book.
